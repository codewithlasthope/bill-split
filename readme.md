# 💰 Bill Splitter

A simple and beginner-friendly Bill Splitter web application built with **HTML**, **CSS**, and **JavaScript**. Enter the total bill amount and the number of people, and the app instantly calculates how much each person needs to pay.

## 📖 How It Works

1. Enter the total bill amount.
2. Enter the number of people sharing the bill.
3. Click the **Split** button.
4. The application calculates and displays how much each person owes.
5. Click **Reset** to clear all fields.

---

## ✅ Input Validation

The application checks that:

- Bill amount is entered.
- Number of people is entered.
- Number of people is greater than 0.
- Invalid or empty inputs display an alert message.

---

## 🧮 Formula Used

```
Amount Per Person = Total Bill ÷ Number of People
```
